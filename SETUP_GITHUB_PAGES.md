# 🚀 Configurar GitHub Pages - KOLFF Dashboard

## ✅ Estado Actual

- ✓ Todos los archivos HTML están en `/docs/`
- ✓ Rama `main` contiene la carpeta `/docs/`
- ✓ Rama `gh-pages` alternativa disponible
- ✓ GitHub Actions workflow listo
- ✓ `.nojekyll` configurado

## 🎯 Próximos Pasos (3 simples clics)

### 1️⃣ Abre GitHub Settings
Ve a: https://github.com/fprdiseno-jpg/kolff-dashboard/settings

### 2️⃣ Busca "Pages" en el menú
- En la barra lateral izquierda
- Haz clic en "Pages"

### 3️⃣ Configura la fuente
En la sección "Build and deployment":
- **Source**: Selecciona `Deploy from a branch`
- **Branch**: Selecciona `main`
- **Folder**: Selecciona `/docs`
- Haz clic en **Save**
