Guía de Uso

Para nueva página o sección:

  1. Copia el CSS de variables (:root)
  2. Copia las clases que necesites (buttons, cards, forms, etc.)
  3. Personaliza colores/espaciado según necesites
  4. Usa clases reutilizables en HTML

Mejores prácticas:

  • Mantén consistencia en espaciado (múltiplos de 8px)
  • Usa variables CSS para cambios globales
  • Combina componentes (ej: btn + icon juntos)
  • Respeta accesibilidad (focus states, contraste)
  • Prueba responsiveness en mobile (375px), tablet (768px), desktop (1440px)

Nombres de clases (BEM light):

  • .btn-primary - Block.modifier
  • .card-title - Block-element
  • .text-muted - Utility class
  • .grid-3 - Variant class
