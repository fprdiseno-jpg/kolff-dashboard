## 🎨 Design System KOLFF — Especificación

### Color System
- **Primary:** #FFB81C (Pantone 1235 C) — Energía, dinamismo
- **Secondary:** #000000 (Pantone Black) — Solidez, contraste
- **Neutros:** 8 tonos de gris (F9F9F9 a 1A1A1A)
- **Estados:** Success, Error, Warning, Info

### Typography
- **Family:** Helvética
- **Weights:** Thin, Light, Roman, Medium, Bold
- **Escala:** H1 (48px) → Caption (12px)
- **Line Height:** 1.17 - 1.5 según tipo

### Components Library
1. **Buttons** — Primary, Secondary, Tertiary (4 variantes)
2. **Inputs** — Text, Select, Checkbox, Radio
3. **Cards** — Base, Hover, Active states
4. **Badges** — Primary, Secondary, Status
5. **Alerts** — Success, Error, Warning, Info
6. **Tables** — Full structure with responsive
7. **Modals** — Dialog, overlay, animations
8. **Icons** — 10 iconos específicos KOLFF
9. **Product Cards** — Energy indicator integration
10. **Service Badges** — Custom KOLFF branding

### Grid & Spacing
- **Base Unit:** 8px
- **Scale:** xs (4px) → 4xl (96px)
- **Breakpoints:** Mobile (4col), Tablet (8col), Desktop (12col)
- **Max Width:** 1280px
